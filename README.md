# GainLog Backend

REST API for GainLog workout tracking app.

## Tech Stack
Node.js, Express, MongoDB, Mongoose, JWT, bcrypt

## API Routes
- POST /auth/register
- POST /auth/login
- GET /api/workouts
- POST /api/workouts
- PATCH /api/workouts/:id
- DELETE /api/workouts/:id
